# Heart Disease Prediction Flask App

This project predicts the possibility of heart disease using machine learning.

## Project Overview

The application takes health-related input values from the user and predicts whether the person may have heart disease.

## Technologies Used

- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML
- CSS

## Files

- app.py → Main Flask application
- model.sav → Trained machine learning model
- heart.csv → Dataset used for training
- templates/ → HTML pages
- static/ → CSS and static files

## How to Run

1. Create virtual environment
2. Install requirements
3. Run:

```bash
python app.py

Note

This project was originally developed in 2021. Running it may require older versions of Python, Flask, NumPy, and Scikit-learn because of dependency compatibility with the saved model.
